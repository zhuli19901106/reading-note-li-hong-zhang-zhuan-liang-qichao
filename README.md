## 写在开头

然而没什么想说的。  

```
入都 其一
丈夫只手把吴钩，意气高于百尺楼。一万年来谁著史? 三千里外欲封侯。
定将捷足随途骥，哪有闲情逐野鸥? 笑指泸沟桥畔月，几人从此到瀛洲!
```

```
入都 其七
一枕邯郸梦醒迟，蓬瀛虽远系人思。出山志在登鳌顶，何日身才入凤池?
诗酒未除名士习，公卿须趁少年时。碧鸡金马寻常事，总要生来福分宜。
```

```
入都 其十
一入都门便到家，征人北上日西斜。槐厅谬赴明经选，桂苑犹虚及第花。
世路恩仇收短剑，人情冷暖验笼纱。倘无驷马高车日，誓不重回故里车。
```
